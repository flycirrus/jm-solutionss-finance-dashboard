# 📊 JM Solutionss - Personal Finance Dashboard

Ein modernes, lokales Dashboard für persönliche Finanzen, entworfen nach den offiziellen **JM Solutionss** Brand Guidelines. 
Behalte den perfekten Überblick über deine Einnahmen, Ausgaben, Budgets und Sparziele.

---

## ✨ Features (MVP)

- 💰 **Key Metrics:** Schneller Überblick über Gesamtsaldo, Einnahmen, Ausgaben und Ersparnisse.
- 📈 **Interaktive Charts:** 
  - Einnahmen-Balkendiagramm (Fix vs. Variabel)
  - Detailreiches Budget-Donut-Diagramm nach Kategorien
- 💳 **Wallet & Karten:** Übersicht deiner gespeicherten Zahlungsmittel.
- 🎯 **Ausgabenlimits:** Verfolge deine monatlichen Limits in Echtzeit mit Fortschrittsbalken.
- 📋 **Transaktions-Management:** Vollständige Historie mit dynamischer Suche und Filterung.
- 🔒 **Lokale Speicherung:** Maximale Privatsphäre! Alle Daten bleiben sicher im `localStorage` deines Browsers.

---

## 🛠️ Technologien

- **Frontend Framework:** React 18 (via Vite)
- **Styling:** Vanilla CSS (Maßgeschneidert nach JM Solutionss CI: `#2E3A8C`, Inter Font)
- **Icons & UI:** Lucide React
- **Datenvisualisierung:** Recharts
- **Routing:** React Router DOM

## 🚀 Installation & Start

Möchtest du das Projekt lokal ausführen? Folge diesen einfachen Schritten:

1. **Repository klonen**
   ```bash
   git clone https://github.com/flycirrus/jm-solutionss-finance-dashboard.git
   cd jm-solutionss-finance-dashboard
   ```

2. **Abhängigkeiten installieren**
   ```bash
   npm install
   ```

3. **Entwicklungsserver starten**
   ```bash
   npm run dev
   ```

4. **Öffnen**
   Besuche `http://localhost:5173` in deinem Browser.
