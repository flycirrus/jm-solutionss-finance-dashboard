# 📊 JM Solutionss - Personal Finance Dashboard

A modern, local-first personal finance dashboard, crafted precisely to the authorized **JM Solutionss** Brand Guidelines. 
Easily track your total balance, income, expenses, budgets, and savings goals at a glance.

---

## ✨ Features (MVP)

- 💰 **Key Metrics:** Quick overview of your Total Balance, Income, Expense, and Savings.
- 📈 **Interactive Charts:** 
  - Income Bar Chart (Fixed vs. Variable)
  - Detailed Budget Donut Chart by category
- 💳 **Wallet & Cards:** Visual overview of your saved payment methods.
- 🎯 **Spending Limits:** Track your monthly limits in real-time with a smart progress bar.
- 📋 **Transaction Management:** Full transaction history featuring dynamic search and status filtering.
- 🔒 **Local Storage:** Maximum privacy! All your financial data is securely stored in your browser's `localStorage`.

---

## 🛠️ Technology Stack

- **Frontend Framework:** React 18 (via Vite)
- **Styling:** Vanilla CSS (Tailored to JM Solutionss CI: `#2E3A8C`, Inter Font)
- **Icons & UI:** Lucide React
- **Data Visualization:** Recharts
- **Routing:** React Router DOM

## 🚀 Installation & Setup

Want to run the project locally? Just follow these straightforward steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/flycirrus/jm-solutionss-finance-dashboard.git
   cd jm-solutionss-finance-dashboard
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Visit `http://localhost:5173` to explore your new dashboard.
